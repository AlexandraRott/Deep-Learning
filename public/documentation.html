Documentation

  • Interaction

    Image classification

    Upon launching the application in Example images tab, the ml5 model is loaded, setting the stage for user interaction. You'll find six initial images displayed, serving as practical examples. Each of these images can be easily classified by clicking the "CLASSIFY" button.

    Following the classification process, users receive clear details about the top match and its confidence level, providing valuable insights into the system's analysis.

    For a more tailored experience, users can opt to upload their own images in Upload image tab by clicking the "Upload image" option in the top bar. After uploading and classifying the image, the system once again provides information about the best match and its confidence level, ensuring a personalized and informative user journey.

    Model training

    The Regression tab provides functionality for training the model with the option to select various parameters. The results are displayed on the corresponding graph, and there is also the possibility to upload a pre-trained model.

    Language Model

    Parameters, such as model type and the number of generated predictions, can be configured at the top. You can write seamlessly in the input field, and predictions will be generated upon clicking the "space" button. Word predictions are presented in the lower area and can be accepted by clicking on them.

  • Frameworks

    Multer Multer is middleware for Express.js, specializing in handling file uploads in web applications. With Multer, users can send files from client-side forms to the server and process them efficiently. It offers features such as file type validation and the ability to change file names. Multer simplifies secure file management on the server and seamlessly integrates into Express.js applications.

    Node.js Node.js is a server-side JavaScript runtime environment based on the V8 JavaScript engine from Google Chrome. It allows the development of high-performance and scalable network applications due to its non-blocking I/O architecture. Node.js is particularly suitable for data-intensive and real-time applications. Developers often utilize Node.js for creating web applications, APIs, and microservices.

    Express.js Express.js is a lightweight and flexible web application framework for Node.js. It is based on simple routing and middleware concepts and provides a robust set of features for developing web and mobile applications. Its modular architecture enables developers to quickly create APIs and RESTful services. The framework also supports a variety of plugins that facilitate the development of Node.js applications.

    TensorFlow.js TensorFlow.js is an open-source JavaScript library developed by Google for training and deploying machine learning models in the browser and Node.js environments. It allows developers to build and train machine learning models directly in JavaScript, leveraging the capabilities of machine learning for tasks like image and text classification, natural language processing, and more. TensorFlow.js facilitates seamless integration of machine learning into web applications, enabling real-time inference and interaction with trained models without the need for server-side processing.

    Ml5.js ml5.js is a JavaScript library for machine learning built on top of TensorFlow.js. It provides developers with easy access to machine learning algorithms without requiring extensive knowledge in the machine learning field. ml5.js offers pre-trained models for various tasks such as image recognition, text analysis, and speech generation. Developers can integrate these models directly into web applications or train and implement custom models. The library is frequently used in the development of interactive creative applications and experiments in the field of artificial intelligence.

    ApexCharts.js ApexCharts.js is a modern JavaScript charting library that offers a wide range of customizable interactive charts. It provides developers with intuitive and feature-rich options for creating visually appealing charts and graphs for web applications. ApexCharts.js supports a variety of chart types, including line, bar, pie, area, and scatter charts. Its ease of use and extensive documentation make it a popular choice for developers seeking to visualize data effectively.

  • Language model implementation

    The implementation process began with the selection of a suitable dataset. I chose the English novel "Oliver Twist" by Dickens. Based on this, I created a Python script that:

    1. Converts the entire text to lowercase
    2. Places each new sentence on a new line
    3. Removes all characters except alphabetic characters and spaces

    As a result, a txt file was obtained with text where all words are in lowercase, separated by spaces, and sentences are separated by line breaks.

    After this preprocessing, I proceeded to develop my models. I wrote a Python script to create a tokenizer in JSON format based on the aforementioned txt file.

    Next, scripts were written by me to create and train two models (RNN and FFNN) on the txt file; they were trained for 10 epochs.

    Since TensorFlow Decision Forest is not supported on Windows, scripts were written to convert the created models to tfjs and run in WSL.

    Then, I wrote a JS script for use in a new section. It loads two models and predicts the next word based on the entered text, considering the user-selected parameters: model type and the number of suggested words.

    Deep learning language models pose a challenge to data privacy due to their reliance on extensive datasets. In my case, there is no inherent risk as the model was trained on classic literature. However, when training models on data containing personal information, applying meticulous anonymization and privacy-preserving technologies becomes crucial. Transparent communication, obtaining informed consent, and ongoing vigilance contribute to a privacy-centric approach.
